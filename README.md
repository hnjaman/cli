## Terminal Commands

### Basic Computer Operation

```
# Rename file  
mv old.txt new.txt
```
### Application running command

```
# Maven Spring boot
mvn spring-boot:run

# Angular
ng s --open

# React
yarn start

# rails 
rails s
```

### Keyboard shortkeys

```
# Open terminal
ctrl + alt + t

# linux terminal new tab
Ctrl + shift + t

# switch between different tabs 
alt + number # number=1,2,3...
```
