### Linux terminal shortkeys

Open terminal
```
ctrl + alt + t
```
Linux terminal new tab
```
Ctrl + shift + t
```
Switch between different tabs 
```
alt + number # number=1,2,3...
```
Zoom in terminal fonts
``` 
ctrl + shift + (+)
```
Zoom out terminal fonts
```
ctrl + (-)
```
